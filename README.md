# 🚀 Matrix Agentic Money System (MAMS)

## Sistema Multi-Agente Autónomo Generador de Ingresos

Sistema de agentes IA colaborativos diseñados para generar ingresos de forma autónoma con mínima intervención humana.

## 🚀 Quick Start

```bash
# 1. Clonar e instalar
pip install -r requirements.txt

# 2. Configurar API keys en .env
cp .env.example .env

# 3. Ejecutar el sistema
python main.py --mode=autonomous

# 4. Monitorear dashboard
python dashboard.py
```
